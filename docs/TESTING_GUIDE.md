# 🚀 快速测试指南 - UI 优化版

## 编译运行

```bash
# 1. 生成 Xcode 项目
xcodegen generate

# 2. 在 Xcode 中打开并运行
open Fuchen.xcodeproj
# 按 ⌘R 运行

# 或者使用 release 脚本编译
./scripts/release-swiftc.sh
open build/Fuchen.app
```

## 🎯 测试重点

### 清理功能 (Clean Tab)

**测试步骤**：
1. 点击顶部"清理"标签
2. 观察 Hero 页面的圆球**呼吸动画**（轻微放大缩小）
3. 点击"预览"按钮
   - ✓ 按钮应立即**微缩反馈**
   - ✓ 0.2秒后页面**流畅切换**
4. 等待扫描完成，观察：
   - ✓ 状态文字："预览完成 — 确认后即可清理"
   - ✓ 卡片**逐个淡入**显示
5. 点击"正式清理"
   - ✓ 弹出授权对话框
   - ✓ 授权后状态文字："清理中，让机器轻装上阵…"
6. 清理完成时：
   - ✓ **绿色横幅从顶部滑入**
   - ✓ ✓ 符号**弹性出现**（带旋转）
   - ✓ 文字："完成 — 您的 Mac 焕然一新"

**预期体验**：
- 每次点击都有即时反馈
- 动画流畅自然
- 文案温暖有情怀

---

### 优化功能 (Optimize Tab)

**测试步骤**：
1. 点击顶部"优化"标签
2. 观察紫色圆球的**呼吸动画**
3. 点击"优化"按钮
   - ✓ 按钮微缩反馈
   - ✓ 立即切换到进度页
4. 观察维护过程：
   - ✓ 状态文字："正在维护，让系统更顺畅…"
   - ✓ 进度指示器动画
   - ✓ 各个维护项**依次出现**
5. 完成时：
   - ✓ **紫色横幅弹出**
   - ✓ ✓ 符号动画
   - ✓ 文字："维护完成 · 已刷新 N 个区域"

## 🐛 已知问题排查

**如果动画不流畅**：
- 检查系统设置 → 辅助功能 → 显示 → "减弱动态效果"是否关闭
- 确保 Mac 处于高性能模式（插电）

**如果看不到呼吸动画**：
- Hero Orb 动画周期为 2 秒，耐心观察
- 动画幅度为 5%，比较微妙

**如果按钮点击无反馈**：
- 检查是否正确编译了最新代码
- 确认 `CleanView.swift` 和 `OptimizeView.swift` 包含 `showStartAnimation` 状态
